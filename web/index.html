GlyphGuard

Erkennt & bereinigt unsichtbare Unicode- und KI-typische Zeichen – 100% lokal.

Impressum GitHub

Was ist das?

Dieses Tool findet unsichtbare Zeichen (Zero-Width, Non-Breaking Space …) und „typische KI-Typographie“ (– — … „ “ ‘ ’) und ersetzt sie durch normale ASCII-Zeichen. Alles passiert lokal im Browser.

JS-Library herunterladen Python (Repo-ZIP)
Hinweis: Die Python-Version liegt im Ordner python/ (CLI inkl.).
